# Node MySQL Todo App

## Install

Some basic Git commands are:

```
$ git clone https://github.com/mohamedsamara/Node-MySQL-Todo-App.git
$ cd project
$ npm install

```

## Start

npm start

## Languages & tools

* [Node](https://nodejs.org/en/)

* [Express](https://expressjs.com/)

* [MySQL](https://www.mysql.com/)
